Course Overview:

Web application programming and hacking course is a course related to full-stack web development. As part of this course apart from full-stack development we also get to learn about best code practices and information security for web application development. It is a over 3 months course that covers topics like client side development, server side development, version control, DevOps, web application security and secure web programming practices. This course is overall a hands-on course where user interface (UI), back-end and database managment skills are applied to construct web applications. As part of the course there are hackathons, hands-on labs, knowldge quizzes, individual and group projects.

Quizzes:

For each lecture provided by the lecturer, a quiz is immedietly conducted to measure the understanding of the students on the topic discussed in the lecture. To answer the quizzes full understanding of the lecture is necessary.

Hands-on Labs:

Every week, based on the lectures provided there is a hands-on lab conducted that would help students apply their understanding to develop small scale applications and its features. Oral lectures containing demonstrations and hands-on labs comprehensively convers all aspects of the course.

Hackathons:

There are a total of four hackathons that would be conducted to check students competencies in cybersecurity aspects of web application development. In these hackathon, there are a multiple levels of hacking challenges that students try to hack. The hackathon also consists of learning defense mechanism to keep away such various level of hacking issues.

Projects:

The course also requires students to create projects individually and as group as well. Through individual projects the students will showcase their course understanding as part of web application. There are a total of two individual projects. After these projects the students will form groups, that would work on a Team project which shall be submitted at the end of the semester. Doing a project in the team would help students gain experience in professional work environments for their future career.
